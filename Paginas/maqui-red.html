Máquinas virtuales y Redes locales

¿Qué son las Máquinas Virtuales?

Una máquina virtual (VM) es un entorno virtualizado que permite ejecutar un sistema operativo dentro de otro. Esto es útil para realizar pruebas, instalar software y experimentar sin poner en riesgo el sistema principal.

  • Permiten simular sistemas operativos.
  • No afectan el sistema anfitrión.
  • Facilitan el aprendizaje y la experimentación.
  • Se pueden eliminar sin dejar rastros en el equipo físico.

Durante el curso instalé Linux en entornos virtualizados, configuré almacenamiento, aumenté memoria RAM, configuré redes internas y probé diferentes herramientas sin afectar mi computadora real.

Redes Locales (LAN)

Una red local (LAN) es un conjunto de dispositivos interconectados dentro de un área limitada, como una casa, salón de clases o empresa. Aprendí los conceptos básicos que permiten la comunicación entre computadoras, como direcciones IP, subredes y protocolos.

Temas aprendidos

  • Asignación de direcciones IP
  • Clases de IP y rangos privados
  • Puertas de enlace y DNS
  • Ping y pruebas de conectividad
  • Topologías de red

FTP (Protocolo de Transferencia de Archivos)

El FTP permite transferir archivos entre computadoras dentro de una red. Instalé un servidor FTP, configuré usuarios, permisos, carpetas públicas y privadas, además de conectarme mediante un cliente FTP desde otra máquina.

  • Creación de usuarios y asignación de contraseñas
  • Permisos de lectura/escritura
  • Transferencia de archivos entre sistemas
  • Conexión desde terminal y desde aplicaciones gráficas

Cisco y Configuración de Routers

Usando Cisco Packet Tracer aprendí cómo funcionan los dispositivos de interconexión en una red real. Pude configurar routers, switches, interfaces y rutas estáticas para asegurar la comunicación entre varias redes.

Comandos básicos que utilicé

  • enable
  • configure terminal
  • interface fastEthernet
  • ip address X.X.X.X 255.255.255.0
  • no shutdown
  • show ip interface brief

Conclusión General

El trabajo con máquinas virtuales y redes locales me permitió comprender cómo funcionan internamente los sistemas, cómo se comunican entre sí y cómo se gestionan los recursos dentro de un entorno informático. Estas habilidades son esenciales para cualquier área relacionada con tecnologías de la información.
